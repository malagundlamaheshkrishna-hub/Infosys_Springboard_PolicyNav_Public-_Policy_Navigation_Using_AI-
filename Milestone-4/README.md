Key Features

1. Admin Dashboard (Management & Analytics)

The Admin panel serves as the command center for the application, providing both control and insight.
-	User Control: Admins can manage the user base by promoting members, locking suspicious accounts, or removing users.

-	Activity Tracking: A live view of active users and a detailed log of system-wide history.

-	Analytics: Interactive charts visualizing model usage, language distribution, and feature popularity.

-	Feedback Analysis: A WordCloud generated from user feedback to identify common sentiments.

-	Data Export: Functionality to download comprehensive reports in CSV/Excel format.




2. User Dashboard & Personalization

Enhanced security and identity features to improve the end-user experience.
-	Security Updates: Users can securely update their email addresses and change account passwords.

-	Identity: Integration of an avatar/DP upload system to personalize the user profile.

-	UI/UX: Refined interface transitions and responsive design elements for better navigation.


Repository Structure

The Milestone4 folder is organized as follows:
-	/src: The fully integrated source code combining Milestone 3 and 4.
-	/notebooks: All relevant .ipynb files used for data processing and model experimentation.
