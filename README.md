# Infosys_Springboard_PolicyNav_Public-_Policy_Navigation_Using_AI-
PolicyNav is an AI-powered system designed to decode and navigate complex public policy. Developed by Malagundla Mahesh Krishna as part of the Infosys Springboard Virtual Internship, this project leverages Machine Learning to transform intricate legislative data into accessible, actionable insights for better civic engagement.

## Description
In this milestone, we developed a secure user authentication system using Streamlit, JWT, and
Ngrok.

## Features Implemented
- User Signup with validation
- Secure Login using JWT
- Dashboard after login
- Forgot Password functionality
- Ngrok integration
  
## How to Run
1. Install dependencies
2. Run Streamlit app
3. Use ngrok to expose the app
